# Cat65 Emulator

this is an emulator for a 6502-based computer system inspired by [Ben Eater](https://www.youtube.com/@BenEater)
that I call the "Cat65" :3c  

documentation for emulator, `.c65` file format and general Cat65 architecture is still in development >w<  

> [!NOTE]  
> it works on my machine but the SDK + JavaFX setup might not work immediately for you
